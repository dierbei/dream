## 1.2 解法

**设钢琴教师带了 5x 名学生，拉丁舞教师带了 6x 名学生**

**那么得出方程：5x + 6x = 76，推出 5x 必为偶数**

**再结合题目说学生数量必须为质数，那么 x 只能为 2（质数中唯一的偶数）**

**那么可得 x = 2，y = 11，最后 4 * 2 + 3 * 11 = 41，答案 D**

**根据题目列出方程：x + y + z = 10; 10x + 8y + 5z = 75**

**进行消元得出：5x + 3y = 25，看出：3y 必须为 5 的倍数，那么 y = 5**

**最终得出 x = 2，答案 B**
